## Features
- [x] Color rendering
- [x] Texture rendering
- [x] Model rendering
- [x] Diffuse rendering
- [x] Transparency rendering

## Build requirements
* C++17 compiler
* [CMake 3.10+](https://cmake.org/)
* [VulkanSDK](https://vulkan.lunarg.com/)
* [OpenSSL](https://www.openssl.org)
* [X11 (Linux only)](https://www.x.org/wiki/)
